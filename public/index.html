Mutations

login

Response

Returns a UserLogin

Arguments
Name Description
account - String!
password - String!

Example

Query
mutation Login(
  $account: String!,
  $password: String!
) {
  login(
    account: $account,
    password: $password
  ) {
    user {
      id
      account
      userName
      birthday
    }
    accessToken
  }
}
Variables
{
  "account": "xyz789",
  "password": "abc123"
}
Response
{
  "data": {
    "login": {
      "user": UserInfo,
      "accessToken": "abc123"
    }
  }
}
